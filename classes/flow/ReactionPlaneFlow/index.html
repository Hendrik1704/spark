ReactionPlaneFlow

class ReactionPlaneFlow.ReactionPlaneFlow(n=2)[source]

This class implements a reaction plane flow analysis algorithm

For this method, the flow is calculated under the assumption that the event plane angle is constant throughout all events. The flow is calculated as

\[v_n = \langle p_T^n \exp{\frac{in\phi_i}{p_T^n}}\rangle\]

where we average over all particles of all events. We return complex numbers, which contain the information about the position of the event plane.

Parameters:
nint, optional

The value of the harmonic. Default is 2.
